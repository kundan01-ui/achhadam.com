🚀 Firebase Firestore Setup

One-click deployment for messaging system

⚠️ Important: Firebase CLI needs browser authentication. Follow these simple steps.

1 Open Command Prompt / Terminal

Navigate to your project folder:

cd C:\Users\Acer\Desktop\Achhadam

2 Login to Firebase

Run this command (it will open browser):

firebase login
  • Browser will open automatically
  • Select your Google account
  • Grant Firebase CLI permissions
  • Return to terminal when done

3 Deploy Firestore Configuration

Run this single command:

firebase deploy --only firestore

This will deploy:

  • ✅ Firestore security rules
  • ✅ 3 composite indexes
  • ✅ Project configuration

4 Wait for Indexes to Build

After deployment, check Firebase Console:

  • Go to: Firebase Console
  • Select project: digital-farming-platform
  • Navigate: Firestore Database → Indexes tab
  • Wait 5-10 minutes for all indexes to show "Enabled"
Open Indexes Page

5 Test Messaging System

After indexes are enabled:

  • Browser 1 (Buyer): Login → Marketplace → Click message icon → Send message
  • Browser 2 (Farmer - Incognito): Login → "Buyer" dropdown → "Buyer Messages"
  • Messages should appear in real-time ✅
✅ Success Indicators:
  • Deployment shows: "✔ Deploy complete!"
  • Indexes show: "Enabled" status (green)
  • Browser console: "✅ Message sent successfully"
  • No 400 errors in console

💡 Troubleshooting:
  • firebase: command not found? Run: npm install -g firebase-tools
  • Login failed? Run: firebase login --reauth
  • Collections not found? Ignore - indexes will be created when first message is sent
  • Still getting 400 errors? Wait for indexes to show "Enabled" status
